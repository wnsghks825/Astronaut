# Astronaut

## 게임 소개
* 한 손으로 할 수 있는 손쉬운 리듬게임
* 위에서 아래로 내려오는 별 모양의 노트들을 획득하는 것이 목표

![MainMenu](./Astronaut/img/그림1.png)

* 깜빡이는 UI는 차례대로 체력과 스킬 게이지
* 체력이 다 떨어지면 게임오버
* 스킬 게이지가 다 차면 돌 모양의 장애물을 없애는 스킬을 사용할 수 있음
